前言：为何要学Python

  1. 当今内卷日益严重的大环境下，似乎掌握一门编程语言成为了一件必须的事情。

  2. Python以其语法简单，容易上手的特点，十分适合作为编程小白的入门级编程语言去学习。完成相同的工作量，别的编程语言或许需要若干步，但Python往往需要只需要很短的步数，并且简短明了。以下是Python和Java对于数组（列表）赋值的对比：

  3. Python作为一门胶水化语言，功能强大，拓展性强，环境搭建相对容易。可以边实践边学习，容易更加熟练掌握。

  4. 以下是几大主流编程语言创始人的对比，仅通过他们发量的差别，相比读者就可以看出来Python这门语言的优越性了！

Python具体可以用来做些什么

  1. 可以用来写一些脚本，代替自己来做一些枯燥而重复的工作，实现自动化办公。（比如从csv或excel表格中读取数据并进行计算，将结果写入新的文件当中）
  2. 借助python强大的第三方库进行数据分析。
  3. 网络爬虫。（笔者就曾批量下载过网络上的数百首音乐和上千章节的小说，只要网速可以，几乎瞬间就可以完成）
  4. 通过Django 或 Falsk 从事web开发。
  5. 凡所应有，无所不有……

学习Python的恰当方式

  1. 认真阅读讲义，理解其行文逻辑。
  2. 文中所展示的代码，强烈建议读者也能够亲自敲写一遍，毕竟，学习编程最重要的是亲自上手去实践，仅仅听理论作用甚微。
  3. 认真思考并完成每节课后练习题。

环境搭建

本文推荐Anaconda+jupyter notebook的环境搭建方式。

  1. 在浏览器上前往此链接：https://www.anaconda.com/products/individual#Downloads

  2. 选择适合自己电脑系统的软件版本

    • 6Windows（64位）环境下选择 64-Bit Graphical Installer

    • MacOS环境下选择 64-Bit Graphical Installer

  3. 安装并运行Anaconda

  4. 在HOME当中选择jupyter notebook并运行，此时浏览器会自动开启，显示类似以下的界面
