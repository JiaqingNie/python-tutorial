1.4 数据类型与类型转换

在一些对数据类型比较严格的编程语言中（例如 C 和 Java），变量在使用前必须声明其类型，并且类型一经声明便不可再修改，以此来提高运行的速度。Python在这方面做了权衡，牺牲了一定的运行速度，换取了更加灵活的语法。在Python中，变量在使用前无需声明其类型，可以直接赋值使用。并且变量的类型可以任意改变。（在理解对象的含义之前，初学者可以暂时这样理解）

当然，有时候我们为了方便计算或后续处理，需要对值进行类型转换，比如将 字符串”6” 转换成 数字6，或者将 浮点型6.66 转换成整数，以便于后续的计算。此时我们需要通过类型转换函数来实现。本节先介绍两个最常见的类型转换函数。
