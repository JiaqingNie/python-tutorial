3.5 字典（dictionary）

如果我们去翻看字典，或者我们想要去查汉字，都需要通过某种检索方式来进行查询，如：部首查字法，音序查字法，等等。这其中便蕴含着键值对的思想，此时的部首或音序就是 键（key），而其对应的汉字，就是值（value），每一个键（key）都有一个值（value）与其对应（虽然在新华字典当中一个部首或音序会对应很多个汉字，但这并不妨碍我们正确理解该思想）。在Python中，同样也有这种数据结构——字典（dict），相比于其他仅仅储存数据本身的数据结构，字典可以同时储存 键 和 值。也可以通过键来查询或修改其对应的值。
